🎯 5-Phase Pipeline 自动化
命令功能备注

--orchestrator --status查看 pipeline 状态 + 当前进度

--rollback-updatePipeline rollback 后一键同步本地状态NEW 替代旧 --sync-state

--rollback-update 详情

Pipeline rollback 后自动同步所有本地文件：

  1. SSH 读 orchestrator_state.json → 确定当前 sub-phase
  2. 更新 bestmodel_phase.json（归档旧 phase → PLANNED）
  3. 归档旧 videos/p/ → _archived/
  4. 清理 docs/github_readme/ 中已归档 phase 的图片
  5. 更新 README.md（表格 + 移除已删除图片引用）

/gpu-train --rollback-update  (自动)    /gpu-train --rollback-update --from p2_coarse  (手动指定)
